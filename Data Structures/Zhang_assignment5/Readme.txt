Part 1 and Part 2 are separated by main files. Each part can be run by
the following commands.
The algorithms for selection, merge, and heap are from the slides.

To compile:
Part 1 - make

Part 2 (test algorithm with n size input) - make main2

To run:
Part 1 - ./main <input file name>

Part 2 - ./main2

To clean:

Part 1 - make clean

Part 2 - make clean2

After done with everything, run clean up on SortingAlgo file - make cleanSort
